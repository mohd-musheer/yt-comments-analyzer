How It Works

  • FastAPI endpoint reads your video URL and extracts the video id.
  • YouTube Data API fetches public top-level comments.
  • Scikit-learn sentiment model predicts Positive/Negative label.
  • Dashboard shows one visual chart plus grouped comment tags.
